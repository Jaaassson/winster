#### 2.2.3 核心模块说明

**路由守卫**
- 全局前置守卫：检查后台路由是否登录
- 动态标题：根据路由设置页面 title
- 语言同步：路由切换时同步 SEO 元信息

**请求封装**
- axios 实例统一配置 baseURL、超时时间
- 请求拦截器：自动添加 JWT Token
- 响应拦截器：统一处理返回格式、错误提示
- 401 自动跳转登录页

**状态管理（Pinia）**
- app store：语言、货币、侧边栏状态
- user store：管理员信息、Token
- product store：产品筛选条件缓存

#### 2.3.2 核心模块说明

**应用工厂模式**
- `create_app()` 函数创建 Flask 应用实例
- 按环境加载配置
- 注册扩展、蓝图、中间件

**统一响应格式**
```python
{
    "code": 200,           # 状态码：200成功，400参数错误，401未认证，403无权限，500服务器错误
    "message": "success",  # 提示信息
    "data": {}             # 数据主体
}
```

**全局异常处理**
- 捕获所有未处理异常
- 统一返回错误响应格式
- 记录错误日志

**分页封装**
- 统一分页参数（page, page_size）
- 统一返回格式（items, total, page, page_size, total_pages）

**多语言字段处理**
- 产品名称、描述等字段支持多语言
- 数据库存储 JSON 格式：`{"zh-CN": "...", "en-US": "..."}`
- 根据请求头 `Accept-Language` 返回对应语言内容

## 三、API 接口设计

### 3.1 接口规范

- 基础路径：`/api/v1`
- 请求方法：RESTful 风格（GET/POST/PUT/DELETE）
- 认证方式：Bearer Token（JWT）
- 数据格式：JSON
- 语言标识：请求头 `Accept-Language: zh-CN / en-US`

## 五、安全设计

### 5.1 认证安全
- JWT Token 认证，有效期 2 小时
- 密码使用 bcrypt 加密（加盐哈希）
- Token 存储在 HttpOnly Cookie 或 localStorage
- 刷新 Token 机制

### 5.2 输入校验
- 后端所有接口参数校验
- SQL 注入防护（使用 ORM）
- XSS 防护（输出转义）
- 文件上传类型/大小校验

### 5.3 其他安全措施
- CORS 跨域白名单配置
- 请求频率限制（可选）
- 敏感操作日志记录
- 错误信息不暴露堆栈详情
