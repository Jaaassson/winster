# 外贸独立站全栈项目 PRD 需求文档

## 一、项目概述

### 1.1 项目背景
本项目为外贸企业打造一套独立站全栈系统，用于展示公司产品、接收海外客户询盘、提升品牌形象。系统采用前后端分离架构，支持多语言、多货币切换，响应式设计适配全球访问。

### 1.2 项目目标
- 建立专业的外贸企业官网，提升品牌形象
- 支持中英文双语切换，USD/CNY 货币切换
- 产品多维度展示，支持分类筛选和搜索
- 在线询盘功能，客户留言自动存入数据库
- 后台管理系统，便于运营人员维护内容
- 响应式设计，适配手机/平板/桌面多终端

### 1.3 目标用户
- **前台用户**：海外采购商、批发商、零售商
- **后台用户**：网站管理员、运营人员、销售团队

---

## 二、功能需求

### 2.1 前台客户功能

#### 2.1.1 多语言与货币
- **语言切换**：支持中文（zh-CN）/英文（en-US）双语切换
- **货币切换**：支持美元（USD）/人民币（CNY）货币切换
- **记忆功能**：用户选择的语言和货币保存到 localStorage
- **SEO 适配**：页面元数据支持多语言

#### 2.1.2 首页
- **轮播 Banner**：3-5 张轮播图，支持后台配置
- **公司简介**：公司介绍文字 + 图片
- **产品分类**：产品分类导航，点击进入对应分类产品列表
- **热销产品**：展示推荐/热销产品（后台配置）
- **工厂展示**：工厂图片展示
- **在线询盘入口**：快速询盘表单入口

#### 2.1.3 产品模块
- **产品列表**：
  - 分类筛选（左侧分类树）
  - 分页功能（每页 12/24/36 个产品）
  - 排序方式（新品、价格升序/降序、销量）
  - 产品卡片（主图、名称、价格、MOQ）
- **产品详情**：
  - 多语言产品名称和描述
  - 产品图集（多图轮播）
  - 规格参数表
  - MOQ（最小起订量）
  - 包装信息
  - 价格（支持货币切换）
  - 立即询盘按钮

#### 2.1.4 询盘表单
- **表单字段**：
  - 客户姓名（必填）
  - 邮箱（必填，邮箱格式校验）
  - 国家/地区（下拉选择）
  - 采购数量
  - 留言内容（必填）
  - 验证码（可选）
- **提交处理**：
  - 前端表单校验
  - 后端数据校验
  - 存入数据库
  - 提交成功提示

#### 2.1.5 静态页面
- **About Us（关于我们）**：公司介绍、发展历程、企业文化
- **Contact（联系我们）**：联系方式、地址、地图、在线表单
- **FAQ（常见问题）**：常见问题解答，折叠面板展示
- **Shipping Policy（运输政策）**：运输方式、时效、费用说明

---

### 2.2 后台管理系统

#### 2.2.1 管理员登录
- **登录方式**：账号 + 密码
- **认证方式**：JWT Token
- **密码加密**：bcrypt 加密存储
- **Token 过期**：支持 Token 刷新
- **权限控制**：路由守卫，未登录跳转登录页

#### 2.2.2 产品管理
- **产品列表**：
  - 分页查询
  - 按分类筛选
  - 按名称搜索
  - 上下架状态筛选
- **新增产品**：
  - 多语言名称（中文/英文）
  - 多语言描述（中文/英文）
  - 产品分类选择
  - 多图上传（主图 + 附图）
  - 规格参数（JSON 格式）
  - 价格（USD/CNY）
  - 库存数量
  - MOQ 最小起订量
  - 包装信息
  - 排序权重
  - 是否上架
- **编辑产品**：同新增产品
- **删除产品**：软删除
- **上下架**：一键切换上架/下架状态

#### 2.2.3 分类管理
- 分类列表（树形结构）
- 新增/编辑/删除分类
- 分类排序
- 多语言分类名称

#### 2.2.4 询盘管理
- **询盘列表**：
  - 分页查询
  - 按时间筛选
  - 按已读/未读筛选
  - 按国家筛选
  - 关键字搜索
- **询盘详情**：查看客户留言详情
- **标记已回复**：标记询盘处理状态
- **导出 CSV**：批量导出询盘数据为 CSV 文件

#### 2.2.5 新闻资讯管理
- 新闻列表（分页、搜索）
- 新增/编辑/删除新闻
- 多语言标题和内容
- 发布/下架
- 封面图上传

#### 2.2.6 网站基础配置
- **联系方式**：电话、邮箱、地址、社交账号
- **首页轮播**：轮播图管理（增删改排序）
- **公司简介**：多语言公司介绍
- **SEO 配置**：网站标题、关键词、描述

## 三、非功能需求

### 3.1 性能要求
- 首屏加载时间 < 3s
- API 响应时间 < 500ms
- 支持并发 100+ 用户访问

### 3.2 安全要求
- 密码加密存储（bcrypt）
- JWT 认证
- SQL 注入防护（ORM）
- XSS 防护
- CORS 跨域配置
- 文件上传安全校验

### 3.3 兼容性要求
- 浏览器：Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- 移动端：iOS Safari、Android Chrome
- 响应式：320px - 1920px 屏幕适配

### 3.4 SEO 要求
- 语义化 HTML
- 页面 TDK（Title、Description、Keywords）
- 多语言 SEO 标签
- 图片 alt 属性
- 结构化数据
